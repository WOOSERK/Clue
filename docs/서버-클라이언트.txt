< 최초 설정 > 
1. 서버는 4명의 클라이언트가 접속하기를 기다림
2. 4명이 접속된면 게임초기설정 정보를 모든 클라이언트에게 전송 
(게임초기설정 정보 : 클라이언트 말 위치 + 범인 정보 + 카드 분배) 


< 주사위정보 전달 과정 >
클 -> 서 : 주사위 값 + 선택정보 + 위치정보 전송
서 -> 클 : 맵 변경 사항을 모든 클라이언트에게 전송


< 추리정보 전달 과정(진실의 방이 아닐 경우) >
서 -> 클 : 추리 요청
클 -> 서 : 추리 정보 전송(추리 정보의 형태 : "장소 + 흉기 + 사람")

서 -> 클 : 전송받은 추리 정보를 모든 클라이언트에게 전송(히스토리 전송)
서 -> 클 : 루프를 돌면서 다른 클라이언트들에게서 단서정보를 받음(단서 정보의 형태 : "있다 + 단서" or "없다 + 정보없음")
"있다" 의 경우 : 서버는 해당 단서를 받아서 해당 턴의 클라이언트에게 전송
"없다" 의 경우 : 다음 클라이언트로 넘어감

만약 루프를 다 돌았는데도 단서를 받지 못할 경우 해당 턴의 클라이언트는 자신의 카드 중 한장을 공개해야함
클 -> 서 : 자신의 카드 중 한장 전송
서 -> 클 : 해당 카드를 모든 클라이언트에게 전송


< 추리정보 전달 과정(진실의 방) >
서 -> 클 : 추리 요청
클 -> 서 : 추리 정보 전송
맞을 경우 : 승리
틀릴 경우 : 죽음
