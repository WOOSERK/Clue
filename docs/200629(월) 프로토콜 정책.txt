패킷 정책

패킷 자료형 char[10], 문자열

교환 패킷 : 총 10바이트
플레이어 식별 + 단계 + 대기 상태 : 1bit + 2bit + 2bit (1바이트 중 5비트)
플레이어 위치 정보 : x y x y, x y x y (2바이트)
카드 정보 : 5bit(2bit는 식별, 3bit는 카드 번호) 5bit 5bit 5bit (3바이트 중 20비트)
주사위 + 선택 정보 : 3bit + 3bit (1바이트 중 6비트)
추리 정보 : 5bit 5bit 5bit (2바이트 중 15비트)
단서 : 1bit(유무 여부) + 5bit(보여줄 카드) (1바이트 중 6비트)


서버 소유 패킷
정답 : 5bit 5bit 5bit(2바이트 중 15비트)


위치 초기 값 세팅 : x : 0x01, y : 0x01
진실의 방 좌표 : x : 0x11, y : 0x11
